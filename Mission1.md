# 마크다운
<hr/>

## 1.1 마크다운 이란?
<hr/>

### 마크다운은 텍스트 기반의 마크업언어로 2004년 존그루버에 의해 만들어졌으며 쉽게 쓰고 읽을 수 있으며 HTML로 변환이 가능하다.

## 1.2 마크 다운의 장-단점
 ### 장점: 
   + 간결하다.
   + 다양한 형태로 변환이 가능하다.
   + 별도의 도구없이 작성가능하다.
   + 지원하는 프로그램과 플랫폼이 다양하다.


### 단점:
+ 표준이 없다.
+ 표준이 없기 때문에 도구에 따라서 변환방식이나 생성물이 다르다.
+ 모든 HTML 마크업을 대신하지 못한다.

# 마크 다운 문법


## 1.헤더(headers)
+ 문서의 제목은 문장 앞에 #을 쓴다.


 # 출력예시: 마크다운

  
+ 문서의 부제목은 문장 앞에 ##을 쓴다.


  ### 출력예시: 
  ## 마크다운


+  #이 1개부터 6개까지 늘어날수록 크기가 줄어든다
## 출력예시:


# 마크다운


## 마크다운


### 마크다운 


#### 마크다운 


##### 마크다운


###### 마크다운


## 2.BlockQuote 



 문자'>'을 사용한다.
 ### 코드예시:
```
> This is a first blockqute.
>	> This is a second blockqute.
>	>	> This is a third blockqute.
```
### 출력예시:

> This is a first blockqute.
>> This is a second blockqute.
>>> This is a third blockqute.
## 3.목록

+ ## 4-1.순서가 있는 목록(번호) 
순서가 있는 목록은 숫자와 그뒷에 점을 찍어 사용한다.

### 코드예시:
```
1. 첫번째
2. 두번째
3. 세번째
```
### 출력예시:
1. 첫번째
2. 두번째
3. 세번째


## 4-2.순서없는 목록(*,+,-사용)

### 코드예시:
```
* 빨강
  + 녹색
    - 파랑
```
### 출력예시:
* 빨강
  + 녹색
    - 파랑
    <hr/>

## 7.링크
+ 글에 링크를 넣는 법은 다음과 같다.
+ 참조 링크
```
 [보이는 주소이름][주소이름]

[주소이름]: URL "주소"   
```
### 출력예시:
링크 : [보이는 주소이름][주소이름]

[주소이름]: URL "주소"
+ 외부링크  
```
사용문법: [주소이름](주소)
적용예: [Google](https://google.com, "google link")
```
+ 자동연결


주소(URL)또는 이메일 주소를 적절한 형식으로 집어 넣는다.
```
* 외부링크: <http://example.com/>
* 이메일링크: <address@example.com>
```
## 8.강조
강조는 *, _, **, __, ~~을 사용한다.
```
*강조*

_강조_

**강조**

__강조__

~~강조~~
```
### 출력예시
*강조*

_강조_

**강조**

__강조__

~~강조~~
## 9.이미지
 + 이미지를 출력하기 위해서는 ```![사진이름](/path/to/img.jpg)``` 이러한 형식으로 입력한다.
 ### 코드예시:
 ```
 ![고양이](image.png)
```
### 출력예시:
![alt text](image.png)
## 10.줄바꿈
+ 줄바꿈을 하기 위해선 " "(띄어쓰기)를 3번 써야한다.
### 코드예시:
```
줄바꿈.
줄바꿈 X 

줄바꿈    .
줄바꿈 O
```
### 출력예시:
줄바꿈
줄바꿈 X 

줄바꿈
줄바꿈 O
